# Radio Lanza

Queremos animarte a lanzar tu producto y vamos a ser tus mentores en este camino. Ya sea montar una startup, abrir un restaurante, o escribir una novela. Este programa te indicará por dónde seguir, qué problemas anticipar, pero sobretodo, te animará a lanzarte.
